📚 Pemahaman Docker

Docker adalah platform containerization yang digunakan untuk menjalankan aplikasi dalam lingkungan yang terisolasi yang disebut container. Container memungkinkan aplikasi berjalan secara konsisten di berbagai environment seperti development, staging, dan production.

Berbeda dengan Virtual Machine, Docker tidak membutuhkan guest OS, sehingga lebih ringan dan cepat dijalankan.

Command Menjalankan Container

docker run -d --name praktikum-web -p 8082:80 praktikum-docker:v1

📄 Dockerfile

FROM nginx:alpine
COPY app/index.html /usr/share/nginx/html/index.html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]

Dockerfile di atas menggunakan image nginx alpine, menyalin file HTML ke dalam container, membuka port 80, dan menjalankan nginx sebagai web server.

🐙 Docker Compose

services:
  web:
    image: nginx:alpine
    ports:
      - "8083:80"
    volumes:
      - ./app:/usr/share/nginx/html:ro

  api:
    image: httpd:alpine
    ports:
      - "8084:80"

  db:
    image: postgres:15-alpine
    environment:
      POSTGRES_USER: praktikum
      POSTGRES_PASSWORD: devops123
      POSTGRES_DB: praktikum_db

Docker Compose digunakan untuk menjalankan beberapa service sekaligus yaitu web, api, dan database dalam satu network.
