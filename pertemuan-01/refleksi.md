\# ✍️ Refleksi Praktikum DevOps



Praktikum DevOps ini memberikan pemahaman awal yang sangat penting mengenai bagaimana perangkat lunak dikembangkan dan dijalankan secara profesional di industri. Saya menyadari bahwa DevOps bukan hanya tentang teknis penggunaan tools, tetapi juga tentang perubahan budaya kerja dan tanggung jawab bersama.



Harapan saya dari praktikum DevOps ini adalah dapat memahami konsep Continuous Integration dan Continuous Delivery secara mendalam. Saya juga ingin menguasai penggunaan Docker untuk containerization serta memahami bagaimana proses deployment dilakukan secara otomatis menggunakan CI/CD pipeline.



Di akhir semester, saya berharap mampu membangun dan mendeploy sebuah aplikasi secara mandiri menggunakan pendekatan DevOps, serta memiliki pemahaman yang kuat mengenai automation, monitoring, dan manajemen infrastruktur modern.
